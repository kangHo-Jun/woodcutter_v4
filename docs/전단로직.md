# 전단 로직 정리

이 문서는 전단(Trim) 옵션 적용 방식과 관련 계산을 간단히 정리합니다.

## 전단 적용 규칙

- 전단 ON 시 유효 높이 계산:
  - `effectiveBoardHeight = boardHeight - trimMargin`
- 전단 ON 시 유효 폭 계산:
  - `effectiveBoardWidth = boardWidth` (변경 없음)

전단 OFF 시에는 원래 판재 크기를 그대로 사용합니다.

## 적용 위치

- 파일: `js/main-unified.js`
- 계산 블록:
  - `trimEnabled`와 `trimMargin` 계산 이후
  - `effectiveBoardWidth`, `effectiveBoardHeight` 계산 부분

## 비고

- 전단은 높이 방향에만 한쪽으로 적용됩니다.
- 다른 로직(패킹, 렌더링, kerf 등)은 변경하지 않습니다.
