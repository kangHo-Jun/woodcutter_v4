# Project History

대장간 프로젝트의 진화 과정과 주요 업데이트 이력입니다.

## [2026.01.30] V4: PDF 고도화 및 실시간 공지 기능
*   **PDF 출력 레이아웃**: 프로젝트 정보 배치 최적화 및 불필요한 라벨 박스 제거로 가독성 향상.
*   **워터마크 시스템**: 부품 위 레이어 배치, Slate Grey 톤 적용, 고급 설정 내 출력 제어 스위치 추가.
*   **실시간 특가 배너 (개발 완료/이번 버전 제외)**: 구글 스프레드시트 연동 및 티커(Ticker) 기능을 개발했으나, 사용자 요청으로 이번 버전 배포에서는 제외 처리했습니다.
*   **UI 문구 정돈**: '나무결 방향 고려' -> '나무결 방향', '트리밍 사용' -> '전단여부'로 레이블 명칭 변경.
*   **브랜딩 정리**: 외부 고정 로고를 워터마크로 통합하여 도면 집중도 개선.

## [2026.01.20 - 2026.01.29] V3: 통합 및 최적화 엔진 강화
*   **로직 통합 (`main-unified.js`)**: PC와 모바일의 핵심 계산 로직을 하나로 통합하여 유지보수성 확보.
*   **라벨링 엔진 도입**: 부품 크기에 따른 자동 알파벳/숫자 라벨링 및 치수 표시 기능.
*   **고급 설정 관리**: localStorage 기반의 설정 저장/불러오기(SettingsManager) 모듈 구현.
*   **원가 계산기**: 판재 두께별 절단 단가 자동 계산 및 상세 내역 리포트 기능.

## [2026.01.10 - 2026.01.15] V2: 모바일 버전 및 사용성 개선
*   **모바일 전용 UI**: 3단계 워크플로우(입력-도면-결과) 및 하단 탭 내비게이션 구현.
*   **테이블 편집**: 부품 목록의 인라인 편집 기능 추가로 데이터 수정 편의성 증대.
*   **반응형 레이아웃**: 다양한 스크린 크기에 대응하는 CSS 유연성 확보.

## [초기 개발 Phase] V1: Core Engine 구축
*   **재단 알고리즘**: Guillotine 및 Free-cut 방식의 2D 패킹 알고리즘 구현.
*   **Canvas 시각화**: 최적화 결과를 실시간으로 캔버스에 드로잉하는 렌더러 구축.
*   **기본 입력 시스템**: 판재 및 부품 정보의 기본 입출력 구조 설계.
