# Woodcutter V4 Project

## Overview
Woodcutter V4 (대장간 V3) is a web-based lumber cutting optimization tool. It uses a modified Guillotine Bin Packing algorithm to minimize waste and provide practical cutting diagrams for table saw operators.

## Tech Stack
- **Frontend**: Vanilla HTML/CSS/JS
- **Core Logic**: `GuillotinePacker` (Custom Bin Packing)
- **PDF Generation**: `jspdf` / `jspdf-autotable`
- **Design**: Modern, responsive UI with glassmorphism and animations.

## Key Features
- Dynamic cutting unit price calculation.
- Optimized cutting diagrams with Kerf (saw blade thickness) consideration.
- Multi-bin packing.
- Mobile/PC responsive design.
